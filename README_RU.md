# Последнее письмо 3D — Godot Desktop

Это **не браузерная игра**. Это обычный desktop-проект для **Godot 4.x**, который можно открыть в движке и собрать в Windows `.exe`.

## Что уже сделано

- 3D-квартира из процедурных примитивов: гостиная, спальня, кухня, ванная, коридор.
- Управление от первого лица.
- Интерактивные объекты: почтовый ящик, фото Анны, зеркало, платье, бутылка, телефон, окно, финальный стол.
- Система писем Анны.
- Выборы в письмах.
- Параметры: `Любовь`, `Вина`, `Безумие`.
- Изменение атмосферы дома в зависимости от выбора игрока.
- Несколько концовок.
- Папки для замены моделей и текстур.
- Подписанный манифест ассетов: `assets/asset_manifest.csv` и `docs/ASSET_MANIFEST.md`.

## Как запустить

1. Установи Godot 4.x.
2. Открой Godot.
3. Нажми **Import**.
4. Выбери файл `project.godot` из этой папки.
5. Открой проект.
6. Нажми **Play**.

Можно также попробовать `run_godot_project.bat`, если Godot добавлен в PATH.

## Управление

- `WASD` — ходить.
- Мышь — осмотр.
- `E` — взаимодействие.
- `Esc` — отпустить/захватить мышь.

## Где менять игру

- Главная сцена: `scenes/Main.tscn`
- Главный скрипт: `scripts/Main.gd`
- Текстуры: `assets/textures/`
- Модели: `assets/models/`
- Список ассетов: `assets/asset_manifest.csv`

## Важно

Интернет-поиск в текущей среде недоступен, поэтому реальные ассеты из интернета я не скачивал. Вместо этого я сделал подписанные слоты и заглушки: ты можешь сказать, какие заменить, а в манифесте уже указано, для чего нужна каждая модель и текстура.
