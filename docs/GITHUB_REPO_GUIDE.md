# Как хранить Godot-игру на GitHub без браузерного сайта

GitHub в этом варианте нужен только как репозиторий с исходниками игры.

## Что загрузить в репозиторий

Загрузи в корень репозитория содержимое папки проекта:

```text
project.godot
scenes/
scripts/
assets/
docs/
README_RU.md
EXPORT_WINDOWS_GUIDE.md
run_godot_project.bat
```

## Что не нужно включать

Не включай GitHub Pages, если тебе не нужна браузерная игра.

Не нужен такой путь:

```text
Settings → Pages
```

Не нужен workflow для деплоя сайта.

## Как работать дальше

1. Открываешь проект в Godot.
2. Меняешь модели/текстуры/скрипты.
3. Коммитишь изменения в GitHub.
4. Собираешь `.exe` локально через Godot.
