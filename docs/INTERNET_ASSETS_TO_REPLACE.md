# Что искать в интернете для замены ассетов

Интернет-поиск здесь недоступен, поэтому реальные файлы из сети не скачаны. Ниже — точные запросы, по которым можно искать бесплатные или коммерчески безопасные ассеты. Проверяй лицензию: лучше `CC0`, `CC-BY` или explicit commercial use allowed.

## Главные модели

1. `old apartment interior 3d model CC0 GLB` — квартира/комнаты.
2. `old mailbox 3d model low poly CC0` — почтовый ящик.
3. `old wooden table 3d model CC0` — финальный стол.
4. `letter stack envelope 3d model CC0` — письма/конверты.
5. `worn sofa 3d model low poly` — старый диван.
6. `old bed 3d model low poly` — кровать.
7. `hanging dress 3d model CC0` — платье Анны.
8. `antique mirror 3d model CC0` — зеркало.
9. `old phone 3d model CC0` — телефон.
10. `photo frame 3d model CC0` — рамка фотографии.
11. `glass bottle 3d model CC0` — бутылка.
12. `old wooden door 3d model CC0` — дверь.
13. `rainy window 3d model texture CC0` — окно с дождём.
14. `candle 3d model CC0` — свечи.
15. `flower vase 3d model CC0` — ваза/цветы для романтической ветки.

## Рекомендованные форматы

- Модели: `.glb`, `.gltf`, `.fbx`, `.obj`.
- Текстуры: `.png`, `.jpg`, `.webp`.
- Для Godot лучше всего использовать `.glb` для моделей и `.png/.jpg` для текстур.
