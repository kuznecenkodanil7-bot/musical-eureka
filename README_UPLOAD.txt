Загрузи эти файлы в репозиторий Godot-проекта:

1) export_presets.cfg -> в корень репозитория рядом с project.godot
2) .github/workflows/main.yml -> именно в папку .github/workflows/

После загрузки открой вкладку Actions, запусти Build Godot Windows EXE.
Готовый архив будет внизу страницы workflow в разделе Artifacts: LastLetter_Windows.
